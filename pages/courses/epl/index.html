Elements of Programming Languages

Elements of Programming Languages is a 20-hour course given by Sergio Decherchi and Emidio Capriotti to the students of the PhD program in Data Science and Computation. The course is running from June 28th to July 5th, 2023 the lectures will be given online. This page will include all the slides and materials of the course.


  • Materials
    Course Schedule
    Introduction to the computer and to programming languages 28 June, 2023.
    Introduction to python and numerical computations 29 June, 2023.
    Exercises 30 June, 2023.
    Introduction to the Linux shell 3 July, 2023.
    Advanced python modules 4 and 5 July, 2023.

    • This is a theoretical and practical course which requires the use of PC.
